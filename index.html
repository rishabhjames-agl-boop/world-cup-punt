How to refill this each matchday →
Edit the DATA object near the bottom of this file. Move finished games into results (add a reads array to settle a game with ✓/✗ chips). For fixtures, a full read takes stakes, statRead, bestBet, valueBet, a legs array (market / pick / conf / note), plus swaps and risk; if you only add a lean string it renders as a quick model card instead. Flags render automatically from the team name. Set each leg's conf 0–100 (dot: green ≥72, amber 60–71, red <60).
These are analytical reads, not guarantees. Every probability is a model estimate built from tournament stats and matchup context — multi-leg bets are longshots by design even when each leg looks probable. Only stake what you're comfortable losing; the edge is in the process, not any single ticket landing.
